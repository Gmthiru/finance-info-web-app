Financial Literacy Website
---------------------------
This is a 3-page website built using HTML, CSS, JavaScript, and XML.

Features:
- Homepage with XML-loaded financial tips
- About page with mission and list
- Contact form with validation
- Light/dark mode toggle
- Responsive layout with media queries
- Accessibility features (alt text, ARIA, keyboard nav)
